初回の起動で、ファイル置き場を選択するダイアログが出てくるので、絶対に「空の」フォルダを選択してください。新しくフォルダを作成することをお勧めします。

また、別IDE内で実行するとうまく動作しません。

## 使い方

### 上のメニューについて

上のメニューには以下の項目があります。

* ファイル
	* コピー　：エディタのプログラム入力部分を、クリップボードにコピーします。
	* ペースト：エディタのプログラム入力部分を、クリップボードの内容に置き換えます。

* 実行
	* コンパイルして実行　：入力したプログラムをコンパイルし、実行します。
	* 前のコンパイルを実行：前回実行したプログラムを、再度実行します。
	* コンソールをクリア　：ウィンドウ下端の入出力部分（後述）に文字がある場合、空にします。
	* 保存先を選択　　　　：初回起動時に設定した保存先を、別のフォルダに変更します。

### プログラム入力部分について

ウィンドウからメニュー、入出力部分（後述）を除いた部分がプログラム入力部分です。プログラムを入力することができます。このバージョンではJavaしか対応していません。

### 入出力部分について

ウィンドウの下端にある、狭い入力エリアのことです。コンソールと同じ役割をしますが、コマンドは実行できません。また、実行前から入力しても、プログラムは認識します。
